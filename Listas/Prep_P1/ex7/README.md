# Pilha Min

Variação de pilha em que as operações de pilha são constantes e tem uma operação para retornar o elemento da pilha com menor chave em tempo constante


# Operações da Pilha

Operações padrão:
- top
- push
- pop

Operação extra:
- bottom: olha a menor chave
